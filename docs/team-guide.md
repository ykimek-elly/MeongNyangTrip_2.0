### 2. 핵심 협업 전략: "인터페이스 우선”

- **UI/UX 디자인 마무리 (팀원 D)**
    - 피그마를 사용하여 메인 홈, 지도, 상세 페이지, 라운지 화면을 최종 확정합니다.
- **데이터 매핑 및 API 명세 확정 (전체 팀)**
    - 확정된 화면을 보며 각 컴포넌트에 필요한 데이터 항목(품종, 나이, 좌표, 코멘트 등)을 정의합니다.
    - 노션에 API 명세서(Request/Response 규격)를 작성하여 백엔드 개발의 '기준점'을 만듭니다.
- **병렬 개발 착수 (A, B, C)**
    - 확정된 인터페이스 규격에 맞춰 각자 인프라 구축, DB 설계, AI 로직 개발을 진행합니다.

### 3. 프로젝트 주요 마일스톤

- **1단계: 인터페이스 확정 및 API 설계** (팀원 D 중심, 전체 검토)
    - 디자인 시스템 확정 및 화면별 데이터 요구사항 정의.
- **2단계: 코어 기능 및 AI 엔진 구축** (팀원 B, C 중심)
    - PostGIS 위치 검색 및 Spring AI 연동 테스트.
- **3단계: 인프라 통합 및 인증 보안** (팀원 A 중심)
    - AWS 실서버 환경 구축 및 로그인/보안 연동.
- **4단계: 전체 통합 및 고도화** (전체 팀)
    - 프론트-백엔드 연동, 카카오 알림톡 자동화 및 최종 성능 테스트.
